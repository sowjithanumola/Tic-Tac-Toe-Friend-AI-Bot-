# TicMaster

TicMaster is a classic Tic Tac Toe game featuring both multiplayer and AI modes. Challenge your friends or play against an unbeatable bot powered by the Minimax algorithm.

## Features

- Play with Friend mode
- Play vs AI Bot mode
- Smart Minimax AI
- Win and Draw detection
- Restart game button
- Responsive design

## Technologies

- HTML
- CSS
- JavaScript

## How to Run

1. Download the project.
2. Open `index.html` in your browser.
3. Choose a game mode.
4. Start playing!
